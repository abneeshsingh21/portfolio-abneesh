### 🌌 Overview
Welcome to my next-generation 3D interactive portfolio. Built natively with **WebGL**, **Three.js**, and **React**, this site completely abandons standard flat 2D scrolling for an immersive, cinematic dive into deep space. 

It acts as the central hub for my flagship developments in Systems Architecture, Artificial Intelligence, and low-level engineering.

### 🛠️ Architecture & Tech Stack
The environment features a unified, heavily optimized 3D scene boasting dynamic lighting, real-time mouse parallax "window" mechanics, bespoke post-processing (including cinematic Bloom, Film Grain, and DSLR Vignettes), and CatmullRom spline-based camera flight paths bridging timeline nodes and floating 3D hologram servers.

### ⚙️ Installation & Development

To deploy the 3D environment locally, initialize the sequence:

```bash
# Clone the repository
git clone https://github.com/abneeshsingh21/portfolio-abneesh.git

# Enter the working directory
cd "portfolio 2.o"

# Install core dependencies
npm install

# Initialize localhost dev server
npm run dev
```
